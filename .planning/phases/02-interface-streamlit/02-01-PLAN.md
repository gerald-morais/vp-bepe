import folium

**Função pública:**
```python
def render_map(df: pd.DataFrame, polygon: Polygon) -> folium.Map:
```
- Criar `m = folium.Map(tiles="OpenStreetMap")` — NÃO passar `location=` nem `zoom_start=` (conflito com fit_bounds)
- Sempre chamar `_add_perimeter(m, polygon)`
- Se `not df.empty`: chamar `_add_route(m, df)`, `_add_markers(m, df)`, e `bounds = _compute_bounds(polygon, df)`
- Se `df.empty`: `bounds = _compute_bounds(polygon)` (apenas o polígono — estado inicial D-01/D-06)
- Chamar `m.fit_bounds(bounds)` e retornar `m`

**Helper _add_perimeter(m, polygon):**
- Converter coords com swap obrigatório: `coords = [(lat, lon) for lon, lat in polygon.exterior.coords]`
- `folium.Polygon(locations=coords, color="#e67e22", weight=3, fill=True, fill_color="#e67e22", fill_opacity=0.08, tooltip="Perímetro autorizado").add_to(m)`

**Helper _add_route(m, df):**
- `df_sorted = df.copy().sort_values(pd.to_datetime(df["Data_Hora"], dayfirst=True).values)` — NÃO: usar `df.copy()` e depois `df_sorted["_sort_key"] = pd.to_datetime(df_sorted["Data_Hora"], dayfirst=True); df_sorted = df_sorted.sort_values("_sort_key").drop(columns=["_sort_key"])`
- `coords = list(zip(df_sorted["Latitude"], df_sorted["Longitude"]))`
- Guard: `if len(coords) < 2: return`
- `folium.PolyLine(locations=coords, color="#2980b9", weight=3, opacity=0.7, tooltip="Rota da VP").add_to(m)`

**Helper _add_markers(m, df):**
- Iterar com `for _, row in df.iterrows()`
- `is_outside = row["Status"] == "OUTSIDE"`
- `folium.CircleMarker(location=[row["Latitude"], row["Longitude"]], radius=6 if is_outside else 4, color="#c0392b" if is_outside else "#27ae60", fill=True, fill_color="#e74c3c" if is_outside else "#2ecc71", fill_opacity=0.9 if is_outside else 0.6, popup=folium.Popup(f"<b>{row['Data_Hora']}</b><br>{row['Endereco']}", max_width=250), tooltip=row["Status"]).add_to(m)`

**Helper _compute_bounds(polygon, df=None):**
- `poly_lons = [c[0] for c in polygon.exterior.coords]`
- `poly_lats = [c[1] for c in polygon.exterior.coords]`
- `min_lat, max_lat = min(poly_lats), max(poly_lats)`
- `min_lon, max_lon = min(poly_lons), max(poly_lons)`
- Se `df is not None and not df.empty`: expandir bounds com `df["Latitude"].min/max()` e `df["Longitude"].min/max()`
- Retornar `[[min_lat, min_lon], [max_lat, max_lon]]`
